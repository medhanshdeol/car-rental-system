# Car Rental Management System

A dynamic web application built using the **WAMP stack** (Windows, Apache, MySQL, PHP). This project allows users to browse a fleet of cars, register/login, and book vehicles, while providing an admin interface to manage the fleet.

## Features
* **User Authentication:** Secure Signup and Login system using password hashing.
* **Dynamic Gallery:** Real-time car listings fetched from a MySQL database.
* **Admin Dashboard:** Interface to add new cars and update rental prices.
* **Checkout System:** Automatic calculation of total costs including security deposits.
* **Booking History:** Users can view their past rental transactions.

## Technology Stack
* **Frontend:** HTML5, CSS3 (Responsive Design)
* **Backend:** PHP (Server-side logic)
* **Database:** MySQL (Relational data storage)
* **Server:** Apache (via WampServer)
